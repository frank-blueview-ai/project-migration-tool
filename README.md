# Project Migrate

`project-migrate` is a modular, extensible migration system designed for developers who need to safely move projects across filesystems, machines, or environments. It provides:

- A Python-based migration engine  
- A CLI tool with subcommands  
- A FastAPI backend for orchestration  
- A React dashboard for visualization  
- Manifest generation  
- Rollback support  
- Parallel copy  
- Logging  
- Docker + Kubernetes deployment options  

This project is built for reliability, transparency, and team-friendly workflows.

---

## Features

### ✔ Migration Engine
- Dry-run previews  
- Real migrations  
- Global + per-project ignore rules  
- Parallel file copying  
- Manifest generation (planned + checksummed)  
- Rollback using manifest  
- Structured logging
